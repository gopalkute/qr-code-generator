# 🔳 QR Code Generator

A simple and fast web application to generate QR codes instantly from text,
URLs, or any custom input. Built with a clean UI and designed for ease of use.

---

## 🚀 Features

* ⚡ Instant QR code generation
* 🔗 Supports URLs, text, and custom data
* 🎨 Clean and minimal user interface
* 📥 Download QR code as an image
* 📱 Responsive design (works on mobile & desktop)

---

## 🛠️ Tech Stack

* React 
* Node.js
* Express js
* tailwind css

## ⚙️ How It Works

1. Enter text or URL
2. Click on "Generate QR Code"
3. Your QR code will be created instantly
4. Download and share it

---

## 💻 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/qr-code-generator.git

# Navigate to project folder
cd qr-code-generator

# Navigate to backend folder and run given commands
cd backend
npm install
npm start

# Navigate to frontend folder and run given commands
cd frontend
npm install
npm run dev

# Open given path in browser
http://localhost:5173/
```
